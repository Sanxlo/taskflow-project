Diseño de la aplicación

El siguiente diseño muestra la interfaz propuesta para TaskFlow.

[Diseño TaskFlow](docs/design/Design.png)

TaskFlow sigue un diseño tipo dashboard minimalista pensado para ser claro y fácil de usar tanto en móvil como en escritorio.

La interfaz incluye:
- Cabecera
- Lista de tareas
- Formulario de tareas
- Panel de estadísticas
