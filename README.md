Diseño de la aplicación

El siguiente diseño muestra la interfaz propuesta para TaskFlow.

[Diseño TaskFlow](docs/design/Design.png)

TaskFlow sigue un diseño tipo dashboard minimalista pensado para ser claro y fácil de usar tanto en móvil como en escritorio.

La interfaz incluye:
- Cabecera
- Lista de tareas
- Formulario de tareas
- Panel de estadísticas

TaskFlow

Aplicación de gestión de tareas desarrollada con HTML, Tailwind CSS y JavaScript.

Demo

Aplicación desplegada en Vercel: https://taskflow-project-iota-gules.vercel.app/

Funcionalidades

- Crear tareas
- Editar tareas
- Eliminar tareas
- Marcar tareas como completadas
- Filtrar por estado
- Buscar tareas
- Persistencia con localStorage
- Modo oscuro
